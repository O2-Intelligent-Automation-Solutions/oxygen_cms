# Roadmap

## Phase 1

Read-only central monitoring and management.

Next milestones:

1. encrypted instance credentials,
2. real connectivity tests,
3. background monitoring/history,
4. OxyGen license/settings/workflow collectors,
5. dashboard/detail polish,
6. Docker/HTTPS deployment hardening.

## Phase 2

Bidirectional management plane:

- remote enrollment,
- webhooks/callbacks,
- persistent outbound instance connection,
- reverse command channel,
- centralized user provisioning,
- notifications/escalations.
